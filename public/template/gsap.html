GSAP STAGGER

This feature was created to test and explore the
various functionalities of GSAP's stagger. This runs on TypeScript.

Play with the options above to see how it affects the animation and the code below. Click any box to have the stagger emanate outward from that specific index number, using the from:[index] syntax.

gsap.to(".box", {
  duration: 1,
  scale: 0.1,
  y: 60,
  ease: "power1.inOut",
  stagger: {
    grid: [5, 13],
    from: "center",
     axis: null,
     ease: Linear.easeNone,
     amount: 1.5
  }
});
